# Security Policy

## Supported Versions

Paperless Genie is pre-1.0; only the latest release receives security fixes:

| Version        | Supported          |
| -------------- | ------------------ |
| latest release | :white_check_mark: |
| older releases | :x:                |

## Reporting a Vulnerability

The bot handles Telegram tokens, Paperless-ngx API tokens, and the contents of
personal document archives — please treat potential vulnerabilities accordingly and
do not report them through public GitHub issues or pull requests.

Instead, please use the **GitHub Private Vulnerability Reporting** feature:

1. On GitHub, navigate to the main page of the repository.
2. Under the repository name, click **Security**.
3. In the left sidebar, click **Advisories**.
4. Click **Report a vulnerability**.

Alternatively, you can contact the maintainer directly at **jegors.cemisovs@gmail.com**.

We will acknowledge your report within 48 hours and coordinate a fix and release
disclosure timeline with you.
